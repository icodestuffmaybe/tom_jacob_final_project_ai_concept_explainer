Course Setup & Prerequisites

To ensure a smooth experience in both the live session and the capstone project, please complete the following setup steps beforehand. This will allow us to focus on building and learning during the class.

1. Account Setup (Required for All)

We will use two main services for our AI models and database. Please create a free account for each:

  • Google AI Studio Account: Go to aistudio.google.com to get your Gemini API Key.
  • Pinecone Account: Sign up for the free "Starter" plan at pinecone.io to get your API key.

2. In-Class Challenge Environment

The in-class challenge is a Python notebook designed to run entirely in your web browser using Google Colab. There is no local software installation required for the live session.

  • Required: A Google Account to access and run the Colab notebook.

3. Capstone Project Environment (Local Setup)

The capstone project is designed to be developed on your own computer. This mimics a real-world development workflow and gives you a project you can run and share from your machine.

  • Code Editor: VS Code or Cursor is highly recommended.
  • Python & Conda: A working installation of Python (version 3.9+) and Conda for managing project dependencies is required.

The RAG Recipe

The RAG process is a six-step pipeline divided into two phases. First, we prepare our data in the one-time **Ingestion** phase. Then, the **Generation** phase runs in real-time to answer user questions.

Phase 1: Ingestion

  • 1. Load: Start with a source document.
  • 2. Split: Break it into smaller text chunks.
  • 3. Embed: Convert each chunk into a vector.
  • 4. Store: Save vectors in a Vector Database.

Phase 2: Generation

  • 5. Retrieve: Find relevant chunks for a query.
  • 6. Generate: Use chunks to create a sourced answer.
